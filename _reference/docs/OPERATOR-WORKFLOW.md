# Operator Workflow

The intended non-technical workflow should be:

1. Log in to the admin panel.
2. Click **Products**.
3. Click **Add Product**.
4. Upload or choose a product image.
5. Enter:
   - Product name
   - Price
   - Short description
   - Full description
   - Category
   - Featured: yes/no
   - Status: draft/active
6. Save.
7. Product appears in the correct public area automatically.
