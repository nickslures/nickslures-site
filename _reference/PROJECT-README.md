# Nick's Lures Website Scaffold

This is a forward-looking starter structure for a fishing lure product website.

It separates:
- public customer pages
- admin/operator pages
- reusable assets
- product data
- future backend/API work
- documentation for non-technical operators

## Important

The admin pages included here are a **prototype shell**. A real website control panel needs a backend, database, login system, and image upload storage. Static HTML alone cannot securely save product changes to the public website.

The structure is designed so you can start simple now and later connect the same pages to Shopify, Square, Supabase, Firebase, WordPress, a custom API, or another backend without starting over.

## Suggested next step

Copy these folders into your existing project carefully. Do not overwrite your current `index.html` unless you have backed it up.
