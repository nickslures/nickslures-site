## Product Label / Spec System
Nick explained that package labels use two rows of options. The selected option is circled with a pen.

Interpretation:
- Top row = weight in ounces
- Bottom row = hook size

Example label:
Top row:
1/8 3/16 1/4 5/16 3/8 oz

Bottom row:
#1 1/0 2/0 3/0 4/0

Important terminology:
The “/0” is pronounced **aught**.
Examples:
- 1/0 = one aught
- 2/0 = two aught
- 3/0 = three aught
- 4/0 = four aught

Do not call it “ot.” It is “aught.”
